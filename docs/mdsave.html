mdsave

Save protein and spin label data from molecular dynamics simulation to a .mat file.

Syntax
mdsave(FileName, MD);
Description

This function saves to a .mat file spin label frame data with and without protein global rotational diffusion, rotation matrices representing the protein's orientation, and spin label side chain dihedral angles data from molecular dynamics (MD) simulation data.

FileName is a string representing the name of the .mat file that contains all of the processed MD simulation data that is required by cardamom to simulate a corresponding EPR spectrum.

MD is a structure containing all output from mdsave, including all raw and processed data that can be used by cardamom to simulate an EPR spectrum. Please see documentation for mdload on proper formatting.

Examples

To save the processed MD simulation data for later use with cardamom, first load and process the MD simulation data using mdload. Then call mdsave using 

AtomInfo.ResName = 'CYR1';

AtomInfo.SegName = 'PROA';

AtomInfo.AtomNames.ONname = 'ON';
AtomInfo.AtomNames.NNname = 'NN';
AtomInfo.AtomNames.C1name = 'C1';
AtomInfo.AtomNames.C2name = 'C2';
AtomInfo.AtomNames.C1Rname = 'C1R';
AtomInfo.AtomNames.C2Rname = 'C2R';
AtomInfo.AtomNames.C1Lname = 'C1L';
AtomInfo.AtomNames.S1Lname = 'S1L';
AtomInfo.AtomNames.SGname = 'SG';
AtomInfo.AtomNames.CBname = 'CB';
AtomInfo.AtomNames.CAname = 'CA';
AtomInfo.AtomNames.Nname = 'N';

MD = mdload('myTraj.DCD',AtomInfo);

mdsave('FrameData.mat',MD);
